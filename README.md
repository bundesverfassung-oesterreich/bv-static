#  Die Entstehung des Bundes-Verfassungsgesetzes 1920
This repository build the static web page for the project „Die Entstehung des Bundes-Verfassungsgesetzes 1920“ an edition of historical documents
* build with [DSE-Static-Cookiecutter](https://github.com/acdh-oeaw/dse-static-cookiecutter)#
* data stem from https://github.com/bundesverfassung-oesterreich/bv-data
